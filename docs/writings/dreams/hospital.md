# 医院，循环，天台
> 2024.11.1日晚
<np>
我与另一人在一个巨大的医院，医院共七层高，每层不知有多宽。但当你走楼梯从七楼再往上走时，你会回到一楼。而据说，当你通过特殊的几个楼梯间从七楼到一楼或从一楼到七楼后，世界会发生难以想象的变化。（似乎是会有鬼）
</np>
<np>
我和那个人因要上厕所便在医院中找厕所，可几乎所有厕所都满人了，我甚至看到一位女士站在男厕所门外，一副十分急着上厕所的样子。
</np><np>
我和他一直下楼，他忽地不见了踪迹，而我在下了十多层楼后在楼梯间的墙壁上发现了几个挂壁式小便器——尽管其竟是铁做的，上面已锈迹斑斑，似乎还有血迹——我马上上完了厕所。之后我便去找他，在七楼大厅发现了他。这时一个人便告诉了我医院的回环秘密，并告诉我我当时“很多白+少量黑”的衣服会使我受到诅咒。我把衣服反了过来，里面是“大部分黑+少量白”，他十分惊讶，说“这能抵御从x$%、”。这时，一个似僵尸的生物冲了过来，我放倒它后，另一人不见了，他消失在了七楼往上的楼梯间中。我也急忙冲了上去。
</np><np>
但我的画面并未随我动，而留在了那里。那人忽地露出险恶的脸，与他的同伙拿出一个东西（电子产品)“点单”了更多的鬼怪，还说：“这个扒皮怎么样?......”
</np><np>
上了七楼后我并未到一楼而真正地到了天台。
</np><np>
这时，这医院的天台竟与这医院格格不入——上面的人尽情地唱着舞着。天台上竟还有一个巨大的田径场！我在人群中穿梭，想从中找到另一人的身影。慢慢我便走到了一个类似准备区的地方。那儿的每个帐篷要似乎都有几人在化妆。我向前走看，看到了像个木乃伊的人、看到了在筒子里扭动身体的人。
</np><np>
之后，我到了废弃区，这里一个人都没有了，这儿的房子破败，似乎以前发生过大事。我一个人蜷缩在角落里面，不禁天真地哼起了歌。
</np><np>
......
</np><np>
爸妈找到了我，我拿手机下类似多邻国的软件，操作了很久很久。手机界面本是乱码，可当梦中的我疑惑了一会儿后，界面中的文字成了我认得的字，只是似乎很多界面的文字组在一起并不能表达出有效信息。
</np>
