# 雪夜，珍视，逃亡
>2024.12.19日晚 完整度：50%
<np>
我和一人躲在一个小房子中，外面风雪交加，室内有点寒冷，我们躲在被子里，我架着台电脑在玩，这时ta说：“做好准备，他们会把我们这里的东西扫荡一空的！”
</np><np>
Ta说着，外面竟似乎真来了个人，他一把推开房门，大喊：“口口口检查口口口。”我把电脑关好，压在身下，在床上装睡。Ta走到床旁的小桌前正想要收起上面的东西，那些东西Ta一碰便不见了。无奈她没碰到最重要的那件便被闯入者拉住了，那人的眼中露出了贪婪的光，我似乎是受不了了，于是便动了手。然后他死了，他消失了、死前喊着“口口会来找你的！”
</np><np>
一会儿后，外面来了一群人。我急忙跳出了小屋。在外奔逃，身外有一人在追着。我跑呀跑，面前的景象不断变幻，可他仍方鲨不名穷追不舍，声音似乎还在耳边。后来我面前出现了一堵铁环组成的墙，呈拱状，我紧忙攀到顶端。本以为他上不来，没想到近8米的高度他一秒便上来了，我一把跳了下去，他下来，我上去，周旋一会儿后我继续逃命。在一家店前，他忽然变成了里面的一个表演者，吸引着无数人——前去一栋破烂的楼中死亡。
</np><np>
我继续逃亡，直到我遇到一群人，他们让我上了车，我牵着只不停啄我的企鹅。与那群人走着，路过那家店时，表演者在表演，我拉着企鹅让它看，它故意装作没看到......
