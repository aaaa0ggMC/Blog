# 无聊，渴望
<np>可能每个人总是会有一段时间精神萎靡，我最近就是这样的。有些人也把这种要死不活的状态叫成emo，然后一些六神无主的悲观人则被美化成了emo“男神”。他们带着一种独特的气质，看起来就十分帅气。这让我想到了肺结核在19世纪被美化的情况。所以我会拒绝emo这个称号，我想探讨的是我这种没精打采对什么事情都感到无聊的状态的另一面——对回忆的渴望。</np>
<np>看着我过去写的、画的，我就感觉这些回忆就在眼前，然后我有点想去触摸，我感受到了一丝魂不守舍的感觉，我又怅惘了。</np>
<np>正如你现在看到的这样，我无聊到不想写下去了。</np>
