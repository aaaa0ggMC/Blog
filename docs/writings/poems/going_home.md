# <p align='center'>夜晚急回老家</p>
<div align='center'>
陌生的山里是一只只白色的眼睛 <br/>
我们驱车深入 <br/>
不是迎接一年的新生 <br/>
却是承受死亡的来临 <br/>
</div>

# 解析
山里路旁立着路灯闪着白光，在我看来，到像是一只只审视的眼睛。<br/>
大姑爷肺癌似乎实在没救了，我们收拾完行李回老家。
三年前似乎也是这样，奶奶肺癌，只不过那时候我在初中学业考试，妈妈瞒着我。<br/>
现在又快过年了，我在几天后和朋友约的爬山也要泡汤了......<br/>
我想说自己好苦啥的，命运多舛啥的，到最后却发现自己挺麻木，无喜无悲<br/>
